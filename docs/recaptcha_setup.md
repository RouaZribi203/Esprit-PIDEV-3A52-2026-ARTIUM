# Google reCAPTCHA configuration for Symfony
# 1. Go to https://www.google.com/recaptcha/admin/create to get your site key and secret key.
# 2. Add these keys to your .env or config/services.yaml as needed.
# 3. Use the site key in your Twig template and the secret key in your controller for server-side validation.

# Example (add to .env.local):
# RECAPTCHA_SITE_KEY=your_site_key_here
# RECAPTCHA_SECRET_KEY=your_secret_key_here
