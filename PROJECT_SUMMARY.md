# ARTIUM Project Summary

ARTIUM is a Symfony-based web platform for managing and promoting artistic and cultural events. It provides features for artists, amateurs, and administrators to create, manage, and participate in events, buy tickets, and interact with the community.

**Key Features:**
1. User authentication and role management (artist, amateur, admin).
2. Event creation and management (date, type, status, gallery, etc.).
3. Ticket purchase and QR code generation for event access.
4. Email notifications with ticket and QR code attachments.
5. Event galleries with location and weather integration.
6. Payment integration (Stripe) for ticket sales.
7. Admin dashboard for event and user moderation.
8. PDF ticket generation with event and user details.
9. Responsive front office for browsing and searching events.
10. Flash messages and notifications for user actions.
11. File uploads for event images and gallery assets.
12. Translations and multi-language support.
13. Secure access to event resources (QR, PDF, etc.).
14. Modular Twig templates for easy UI customization.
15. Doctrine ORM for database management.
16. RESTful routes for event and ticket operations.
17. Weather API integration for event day forecast.
18. Example: An artist creates an event, uploads images, sets ticket price; users buy tickets, receive QR by email, and download PDF.
19. Example: Admin reviews events, manages users, and monitors ticket sales.
20. Example: User views event details, checks weather, buys ticket, and receives confirmation.

This project is designed for scalability, security, and a seamless user experience in the cultural sector.
